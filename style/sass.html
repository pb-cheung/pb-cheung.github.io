# Sass

# 1.变量

# 用处

属性值定义为一个具名变量后更有语义化

  • $开头
  • 变量名中的单词以连字符（-）和下划线（_），（$highlight-color == $highlight_color）
  • 作用域：花括号（代码块）内

# 2.嵌套（Nesting）

# 用处

HTML有嵌套的层级结构。但是CSS写样式却不能嵌套式书写。必须外层选择器 内存选择器 目标元素这样书写， 层级较深的情况，较外层的会被重复书写。

解决嵌套层级较深的元素，针对其写样式规则，较外层的选择器要重复书写

#content {
  article {
    h1 { color: #333 }
    p { margin-bottom: 1.4em }
  }
  aside { background-color: #EEE }
}

 /* 编译后 */
#content article h1 { color: #333 }
#content article p { margin-bottom: 1.4em }
#content aside { background-color: #EEE }

# 父选择器（&）

# 用处

在嵌套中

  • 伪类的书写
  • （父选择器左侧还有选择器）属性选择器的书写
.alert {
  // The parent selector can be used to add pseudo-classes to the outer
  // selector.
  &:hover {
    font-weight: bold;
  }

  // It can also be used to style the outer selector in a certain context, such
  // as a body set to use a right-to-left language.
  [dir=rtl] & {
    margin-left: 0;
    margin-right: 10px;
  }

  // You can even use it as an argument to pseudo-class selectors.
  :not(&) {
    opacity: 0.8;
  }

  // 适配IE浏览器
  #content aside {
    color: red;
    body.ie & { color: green }
  }
}

# 选择器列表（Selector Lists）

选择器列表：多个选择器使用相同的样式，用逗号分隔符分开书写。

h1, h2, h3 {
  margin: 0;
}
.container {
  h1, h2, h3 {margin-bottom: .8em}
}

h1, h2, h3 {
    margin: 0;
}
.container h1, .container h2, .container h3 {
  margin-bottom: .8em;
}

# 嵌套属性

连字符号-分割处，后半部分的子属性可以放在一个花括号{}内。

nav {
  border: {
    style: solid;
    width: 1px;
    color: #ccc;
  }
}

nav {
  border-style: solid;
  border-width: 1px;
  border-color: #ccc;
}

# 3.模块化

  • 模块命名，以下划线（_）开头
  • 模块使用，先引入@use module-path，再以文件名为命名空间使用模块中内容（变量、函数、mixin）
// _base.scss
$font-stack: Helvetica, sans-serif;
$primary-color: #333;

body {
  font: 100% $font-stack;
  color: $primary-color;
}


// styles.scss
@use 'base';

.inverse {
  background-color: base.$primary-color;
  color: white;
}
