# 垂直居中

# 1.flex实现方案

在父容器上分别添加以下属性：

.wrapper {
    display: flex;
    justify-content: center;
    align-items: center;
}

# 2.absolute+margin:auto

<style>
  .wp {
      position: relative;
      background-color: #ccc;
      width: 300px;
      height: 300px;
  }
  .box {
    width: 100px;
    height: 100px;
    background-color: grey;
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    margin: auto;
  }
</style>
<div class="wp">
  <div class="box size">123123</div>
</div>

# 解释：

  • 需要设置子元素尺寸；
  • 当一个元素被绝对定位后，它的margin属性值可以被解释为相对于父元素宽度和高度的偏移量;
  • 子元素上top:0; left:0; right:0; bottom:0;将四个边缘与父元素对齐，占据了整个父元素的空间；

# 参考

https://www.hongkiat.com/blog/css-margin-auto/

# 3.absolute+负margin

需要知道子元素尺寸

.wp {
    position: relative;
}
.box {
    position: absolute;;
    top: 50%;
    left: 50%;
    margin-left: -50px;
    margin-top: -50px;
}

top: 50%;就可以实现垂直居中，但是绝对定位是基于子元素的左上角，我们期望的是子元素的中心垂直居中。

进行修正：借助负外边距，让元素方向定位，设置上外边距取值为子元素高度的一半，即可实现。

参考： https://juejin.cn/post/6844903679242305544

# 4.absolute+transform

和3.思路类似

.parent {
  position: relative;
}

.child {
  position: absolute;
  top: 50%;
  transform: translateY(-50%); /* 垂直居中子元素 */
}

# 5.表格css-table

tabel单元格中的内容天然就是垂直居中的，只要添加一个水平居中属性就可以了。但是table布局现在很少被使用，推荐使用table属性。

<style>
.wp {
    display: table-cell;
    text-align: center;
    vertical-align: middle;
    width: 300px;
    height: 300px;
    background-color: #ccc;
}
.box {
    display: inline-block;
    background-color: grey;
}
</style>
<div class="wp">
  <div class="box size">123123</div>
</div>

# 解释：

  • top:50%;把子元素定位到距离父元素高度一半的位置
  • transform: translateY(-50%);使用CSS的2D变换将元素上移自身元素的一半。

当我们设置一个元素的transform属性时，它会创建一个新的坐标系统，其中元素相对于原始位置进行定位。

# margin:auto为什么不能实现垂直居中？

我们知道，块级元素即使设置了宽度，也会占满一行，为什么会这样?

因为默认的宽度规则是“适应于父级”规则（在水平方向上自动扩充）。即

margin-left+border-left-width+padding-left+width+padding-right+border-right-width+margin-right= width of containing block

对于绝对定位元素，有以下算式：

left+margin-left+border-left-width+padding-left+width+padding-right+border-right-width+margin-right+right= width of containing block

而auto的作用是：自动填充剩余空间， 所以给div设置margin：auto时，在水平方向上 margin会填充 div这一行中除了left+border-left-width+padding-left+width+padding-right+border-right-width+right的剩余的空间，当左右的margin都设置为auto时，会平分剩余空间，从而实现水平居中。那为什么没有按照上述同样的方式填充垂直方向上的剩余空间呢？

因为在垂直方向上，块级元素不会自动扩充，它的外部尺寸没有自动充满父元素，也没有剩余空间可说。所以margin：auto不能实现垂直居中。
