# 立即调用函数表达式

Immediately Invoke Function Expression

语法：

(function() {})()

第一个圆括号，圆括号运算符函数声明，匿名函数变成了一个表达（js中不允许声明匿名函数） 第二个圆括号，是函数调用

# 函数声明

js中允许的几种函数声明方式，匿名函数声明了必须要赋给一个变量，只声明会报错。

function foo() {} // 声明具名函数

let foo = function() {} // 匿名函数表达式

let foo = function fn() {} // 函数声明+赋值表达式

function() {} // Uncaught SyntaxError: Function statements require a function name

# 其他

怎么理解(function(){}())？

先看具名函数的情况

() // Uncaught SyntaxError: Unexpected token ')'
function fn(){} () // Uncaught SyntaxError: Unexpected token ')'
(function fn(){}())

解释： 第二行，具名函数声明 + 圆括号运算符，声明语句ok，圆括号运算符执行报语法错误 第三行，外层有个圆括号，括号内变为表达式，声明语句返回函数体，然后执行函数调用
