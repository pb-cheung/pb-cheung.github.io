# 浮点数运算

# 0.1+0.2!==0.3

0.1 和 0.2 都转化成二进制后再进行运算

0.0001100110011001100110011001100110011001100110011001101 +
0.001100110011001100110011001100110011001100110011001101 =
0.0100110011001100110011001100110011001100110011001100111

结果转成十进制是 0.30000000000000004，印证了截图中的第二个表达式。

返回值：使用定点表示法表示给定数字的字符串。

# 四舍五入

错误：1.355 => 1.35，千分位的5进位得到1.36

实际是：1.355按照双精度浮点数规范对应的二进制数在千分位进行四舍五入

1.355.toFixed(2) // '1.35' 并不是期望中的1.36
1.355.toPrecision(20) // '1.3549999999999999822'

# 解决方法

加减乘除四则运算，思路是：小数转换成整数再进行运算。

推荐使用number-precision (opens new window)这个库。

例如：

/**
 * 加法运算
 */
function add(num1, num2) {
    const num1Digits = (num1.toString().split('.')[1] || '').length;
    const num2Digits = (num2.toString().split('.')[1] || '').length;
    const baseNum = Math.pow(10, Math.max(num1Digits, num2Digits));
    return (num1 * baseNum + num2 * baseNum) / baseNum;
}

/**
 *  乘运算，避免浮点数运算出错
 */
function accMul(arg1, arg2) {
  let m = 0
  let s1 = arg1.toString()
  let s2 = arg2.toString()
  try {
    m += s1.split('.').length > 1 && s1.split('.')[1].length
  } catch (e) {
    console.warn(e)
  }
  try {
    m += s2.split('.').length > 1 && s2.split('.')[1].length
  } catch (e) {
    console.warn(e)
  }
  return (Number(s1.replace('.', '')) * Number(s2.replace('.', ''))) / Math.pow(10, m)
}
