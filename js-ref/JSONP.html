# JSONP

过程：

  1. 客户端准备一个接收数据的全局函数
  2. 浏览器解析到外联script标签，发出请求
  3. 服务端收到请求和对应的回调函数名，返回携带数据的函数调用
  4. 客户端执行调用

前端代码：

<html>
	....
	<script>		
		window.myCallback = (res) => { // 接收响应数据的回调函数
    		console.log(res)
		}
	</script>
    <script url="xxx?callback=myCallback">
    			// script标签的请求必须在写在定义全局函数之后
    			// 这里需将前端接收响应数据的回调函数函数名传递给后端
    			// 这里callback这个键名是前后端约定好的
	</script>
	</body>
</html>

后端返回：

myCallback({ 
    // 上段代码中script标签中内容，一个函数的调用，入参即前端请求的数据将数据。
    // 加载到前端后执行
	name:'lihua',
    age: 18
})
