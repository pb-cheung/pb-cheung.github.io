# 闭包（closures）

# 定义

闭包是：（内部）函数+其作用域（函数声明的的时候的状态=词法环境-lexical environment）。 一个闭包提供了通过内部函数访问外部函数作用域的途径。

A closure is the combination of a function bundled together (enclosed) with references to its surrounding state (the lexical environment). In other words, a closure gives you access to an outer function's scope from an inner function. In JavaScript, closures are created every time a function is created, at function creation time.

# 举例说明

function makeFunc() {
  const name = "Mozilla";
  function displayName() {
    console.log(name);
  }
  return displayName;
}

const myFunc = makeFunc();
myFunc();

在其他编程语言中，函数中的局部变量只在函数执行期间存在，makeFunc()执行完毕后，name变量不再可备访问。这在JavaScript中是可行的，myFunc()可以正常执行。

原因：

  • js中的函数会形成闭包（函数+词法环境）。
  • 词法环境由闭包创建时作用域内的所有局部变量组成。
  • myFunc是makeFunc运行时displayName函数实例的引用
  • displayName保存了它词法环境的引用，这个词法环境中包含局部变量name
  • myFunc被调用，变量name仍然是可以访问到的，值创给console.log

# 作用

# 1.隔离作用域

ES6之前，JS没有块级作用域，只有函数作用域。

作为独立模块存在，防止命名冲突

var ns = ns || {};

(function (ns){
  ns.name = 'Tom';
  ns.greet = function(){
    console.log('hello!');
  }
})(ns);
console.log(ns); // { name: 'Tom', greet: [Function] }

# 2.模拟私有方法

JS语言中没有原生的private之类的支持。

使用闭包来定义公共函数，也叫模块模式。

var Counter = (function() {
  var privateCounter = 0;
  function changeBy(val) {
    privateCounter += val;
  }
  return {
    increment: function() {
      changeBy(1);
    },
    decrement: function() {
      changeBy(-1);
    },
    value: function() {
      return privateCounter;
    }
  }
})();

console.log(Counter.value()); /* logs 0 */
Counter.increment();
Counter.increment();
console.log(Counter.value()); /* logs 2 */
Counter.decrement();
console.log(Counter.value()); /* logs 1 */

# 常见题目

# 题目一

for (var i = 1; i <= 5; i++) {
  setTimeout(function timer() {
    console.log(i)
  }, i * 1000)
}

预期输出是：

1
2
3
4
5

实际输出是：

6
6
6
6
6

解释：

  • 通过for循环+setTimeout创建了5个异步回调函数
  • 根据事件循环机制，异步函数会在同步函数执行完毕之后再执行，即for循环先执行完毕（此时i=6），然后回调函数按声明顺序依次执行
  • 回调函数timer执行，其中的变量i，沿着作用域链找到变量i此时的值为6，因此打印5个6

# 闭包解决

for (var i = 1; i <= 5; i++) {
  (function(j) {
    setTimeout(function timer() {
      console.log(j)
      }, j * 1000)
  })(i)
}

解释：

  • setTimeout包裹了一层立即执行函数，函数接收i赋给形参j
  • 同样的先执行同步代码，for循环和立即执行函数代码会先执行，然后一次执行各个timer
  • 此时用到异步函数闭包，j变量在timer函数的词法环境内，它的值为异步函数声明时候传入的值没有被修改过，分别为（1/2/3/4/5），因此会分别打印出来（1/2/3/4/5）

# let替换var解法

for (let i = 1; i <= 5; i++) {
  setTimeout(function timer() {
    console.log(i)
  }, i * 1000)
}
  • var是函数作用域，包括for循环的函数内都可访问到
  • let是块作用域，for循环每次迭代都会创建一个新的块级作用域，每个迭代的i都会在块级作用域内创建一个新的绑定，每个迭代中的变量i都是一个新的变量
  • 回调函数是一个闭包，它可以访问它词法环境中的变量i，分别为1/2/3/4/5
