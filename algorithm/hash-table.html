# 一、有效的字母异位词

力扣 242. 有效的字母异位词 (opens new window)

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。
注意：若 s 和 t 中每个字符出现的次数都相同，则称 s 和 t 互为字母异位词。
示例 1:
输入: s = "anagram", t = "nagaram"
输出: true
示例 2:
输入: s = "rat", t = "car"
输出: false
提示:
1 <= s.length, t.length <= 5 * 104
s 和 t 仅包含小写字母
进阶: 如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？

# 思路

  • 先遍历第一个字符串，建立以字母为key，字母出现数量为value的哈希表
  • 遍历第二个字符串
    • 如果出现表中没有的直接返回false
    • 如果表中有，将value值减1
    • 最后，哈希表中如果还有值大于0的，那就是第二个字符串的字符少
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isAnagram = function(s, t) {
    let map = new Map();
    for (let i = 0; i < s.length; i++) {
        const val = s[i];
        if (map.has(val)) {
            map.set(val, map.get(val) + 1);
        } else {
            map.set(val, 1);
        }
    };
    for (let i = 0; i < t.length; i++) {
        const val = t[i];
        if (!map.has(val) || map.get(val) === 0) {
            return false;
        } else {
            map.set(val, map.get(val) - 1);
        }
    }
    return [...map.values()].some(num => num > 0) ? false: true;
};

# 优化

  • 先判断字符串长度是否相等，就能排除一部分，且思路中2.3就不用做了
  • 可以使用数组，下标默认就是字符的Unicode编码值，隐式的用来当做key
/**
 * 力扣 242. 有效的字母异位词
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isAnagram = function(s, t) {
    if (s.length !== t.length) {
        return false;
    }
    const ret = new Array(26).fill(0);
    const base = 'a'.charCodeAt();
    for (const word of s) {
        ret[word.charCodeAt() - base] ++;
    }
    for (const word of t) {
        if (ret[word.charCodeAt() - base] <= 0) {
            return false;
        }
        ret[word.charCodeAt() - base] --;
    }
    return true;
};

const s = 'anagram';
const t = 'nagaram';

console.log(isAnagram(s, t));

# 二、两个数组的交集

力扣349 两个数组的交集 (opens new window)

给定两个数组 nums1 和 nums2 ，返回 它们的交集 。输出结果中的每个元素一定是 唯一 的。我们可以 不考虑输出结果的顺序 。
示例 1：
输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2]
示例 2：
输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[9,4]
解释：[4,9] 也是可通过的

# 思路

  • 因为重复的元素输出只输出一个，所以需要去重，利用Set哈希表的特点，自动去重
  • 去重后的set用展开运算符转成数组再过滤，过滤传入回调函数是另一个哈希表中是否包含当前项

# 代码

/**
 * @param {number[]} nums1
 * @param {number[]} nums2
 * @return {number[]}
 */
var intersection = function(nums1, nums2) {
    let set1 = new Set(nums1);
    let set2 = new Set(nums2);

    return [...set1].filter(num => set2.has(num));
};

# 四、两数之和

力扣1. 两数之和 (opens new window)

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。
你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。
你可以按任意顺序返回答案。
示例 1：
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。
示例 2：
输入：nums = [3,2,4], target = 6
输出：[1,2]
示例 3：
输入：nums = [3,3], target = 6
输出：[0,1]
提示：
2 <= nums.length <= 104
-109 <= nums[i] <= 109
-109 <= target <= 109
只会存在一个有效答案
进阶：你可以想出一个时间复杂度小于 O(n2) 的算法吗？

# 思路

  • 双重循环，暴力解法，时间复杂度O(n^2)
  • 使用哈希表
    • 遍历数组元素，根据当前元素的值，查询哈希表中是否有目标值
    • 如果有直接返回（题目假设只有一对满足）
    • 如果没有将当前值放入哈希表，元素值为key，元素下标为value

# 代码

/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function(nums, target) {
    let map = new Map(); // 数字为key，下标为val
    for (let i = 0; i < nums.length; i ++) {
        const num1 = nums[i];
        const num2 = target - num1;
        if (map.has(num2)) {
            return [map.get(num2), i];
        } else {
            map.set(num1, i);
        }
    }
    return [];
};

# 五、四数相加

力扣454. 四数相加 (opens new window)

给你四个整数数组 nums1、nums2、nums3 和 nums4 ，数组长度都是 n ，请你计算有多少个元组 (i, j, k, l) 能满足：
0 <= i, j, k, l < n
nums1[i] + nums2[j] + nums3[k] + nums4[l] == 0
示例 1：
输入：nums1 = [1,2], nums2 = [-2,-1], nums3 = [-1,2], nums4 = [0,2]
输出：2
解释：
两个元组如下：

  1. (0, 0, 0, 1) -> nums1[0] + nums2[0] + nums3[0] + nums4[1] = 1 + (-2) + (-1) + 2 = 0
  2. (1, 1, 0, 0) -> nums1[1] + nums2[1] + nums3[0] + nums4[0] = 2 + (-1) + (-1) + 0 = 0
    示例 2：
    输入：nums1 = [0], nums2 = [0], nums3 = [0], nums4 = [0]
    输出：1

# 思路

  • 暴力法，四重循环……，时间复杂度太高
  • 问题拆分+哈希表
    • 前两个数组、后两个数组分为两组，双重循环遍历所有两个数组元素和
    • 建立以“和”为key，同一和的组合的数目为val的哈希表，这样就有了两个哈希表
    • 遍历其中一个哈希表，检查另一个哈希表中是否有满足两个和的和为0的key值，若果有将两者的组合数目相乘做累加

# 代码

/**
 * @param {number[]} nums1
 * @param {number[]} nums2
 * @param {number[]} nums3
 * @param {number[]} nums4
 * @return {number}
 */
var fourSumCount = function(nums1, nums2, nums3, nums4) {
    const mapFirst = new Map();
    for (let i = 0; i < nums1.length; i++) {
        for (let j = 0; j < nums2.length; j++) {
            const sum = nums1[i] + nums2[j];
            if (mapFirst.has(sum)) {
                mapFirst.set(sum, mapFirst.get(sum) + 1);
            } else {
                mapFirst.set(sum, 1);
            }
        }
    }
    const mapLast = new Map();
    for (let i = 0; i < nums3.length; i++) {
        for (let j = 0; j < nums4.length; j++) {
            const sum = nums3[i] + nums4[j];
            if (mapLast.has(sum)) {
                mapLast.set(sum, mapLast.get(sum) + 1);
            } else {
                mapLast.set(sum, 1);
            }
        }
    }
    let count = 0;
    [...mapFirst.keys()].forEach(keyFirst => {
        const keyLast = 0 - keyFirst;
        if (mapLast.has(keyLast)) {
            count += mapFirst.get(keyFirst) * mapLast.get(keyLast);
        }
    });
    return count;
};

# 六、赎金信

力扣383. 赎金信 (opens new window)

给你两个字符串：ransomNote 和 magazine ，判断 ransomNote 能不能由 magazine 里面的字符构成。 如果可以，返回 true ；否则返回 false 。 magazine 中的每个字符只能在 ransomNote 中使用一次。 示例 1： 输入：ransomNote = "a", magazine = "b" 输出：false 示例 2： 输入：ransomNote = "aa", magazine = "ab" 输出：false 示例 3： 输入：ransomNote = "aa", magazine = "aab" 输出：true 提示： 1 <= ransomNote.length, magazine.length <= 105 ransomNote 和 magazine 由小写英文字母组成

# 思路

  • 1.双重循环，暴力法
  • 2.一重循环+哈希表
    • 遍历提供字符的字符串，过程中建立哈希表，字符为key，字符出现数量为value
    • 然后再遍历目标字符串，如果哈希表中有，则将字符数量减一，继续遍历，如果没有返回false
  • 3.和2.类似，区别是，使用数组创建哈希表，因为题目中说了字符全部由小写英文字母组成，
    • 这样就可以创建一个连续空间，长度为小写字母个数，初始化为0
    • 哈希的key是字母对应的Unicode，字符出现的次数是value

# 代码

/**
 * @param {string} ransomNote
 * @param {string} magazine
 * @return {boolean}
 */
var canConstruct = function(ransomNote, magazine) {
    if (ransomNote.length > magazine.length) {
        return false;
    }
    const map = Array(26).fill(0);
    const offset = 'a'.charCodeAt();
    [...magazine].forEach(ele => {
        const index = ele.charCodeAt() - offset;
        map[index] += 1;
    });
    const result = [...ransomNote].filter(ele => {
        const index = ele.charCodeAt() - offset;
        if (map[index] === 0) {
            return true;
        } else {
            map[index] -= 1;
            return false;
        }
    });
    return result.length > 0 ? false: true;
};

# 七、三数之和

(opens new window)

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请

你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。
示例 1：
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
注意，输出的顺序和三元组的顺序并不重要。
示例 2：
输入：nums = [0,1,1]
输出：[]
解释：唯一可能的三元组和不为 0 。
示例 3：
输入：nums = [0,0,0]
输出：[[0,0,0]]
解释：唯一可能的三元组和为 0 。

# 思路

  • 暴力解法，三重循环
  • 一重循环 + 双指针（题目要求返回是组合情况，而不是下标，所以可以对给定数组进行排序，才能使用双指针）
    • 升序排序
    • 初始化，遍历索引的范围[0, len-3]，len-2,len-1要留给两个指针指向的元素
    • 开始遍历，left初始为第二个元素，right指向最后一个元素，获取当前三个指针指向的元素的和
    • 和如果大于0，right向左移动
    • 和小于0，left右移
    • 等于0，放入结果中

# 代码

var threeSum = function(nums) {
    let result = [];
    nums.sort((a,b) => a - b);
    for (let i = 0; i < nums.length - 2; i++) {
        if (nums[i] > 0) break; // 第一个数大于0，则三数相加必然大于0，不符合
        if(i > 0 && nums[i] === nums[i-1]) continue; // 去重
        let L = i + 1;
        let R = nums.length - 1;
        while(L < R) {
            const sum = nums[L] + nums[i] + nums[R];
            if (sum === 0) {
                result.push([nums[L], nums[i], nums[R]]);
                while(L < R && nums[L] === nums[L + 1]) L++; // 去重（相邻相同元素）
                while(L < R && nums[R] === nums[R - 1]) R--; // 去重
                L ++;
                R --;
            } else if (sum < 0) {
                L ++;
            } else if (sum > 0) {
                R --;
            }
        }
    }
    return result;
}
