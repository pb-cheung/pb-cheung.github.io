# 常用字符串方法

# 查找

# 返回索引

  • indexOf()，返回第一个匹配字符的索引，未匹配到返回-1
  • lastIndexOf()，返回最后一个匹配字符的索引

# 返回元素

  • charAt(index)
    • 返回索引index对应的字符
    • 如果index超出范围(字符串长度减1)，charAt()返回一个空字符串，而方括号表示法返回undefined

# 返回Unicode码点

  • charCodeAt(index)，字符串中索引index对应字符的Unicode编码值
  • codePointAt(index)，字符串中索引index对应字符的完整Unicode编码值，遇到特殊字符使用这个

# 根据字符顺序排序

let str = '4kd0qouns8e463401gsaqwm';
str.split('').sort((a,b) => a.charCodeAt(0) - b.charCodeAt(0));
console.log(str);

# 截取

  • slice(indexStart, indexEnd)
    • 参数取负值，表示从结尾倒数
  • substring(indexStart, indexEnd)
    • 不改变原字符串，返回从indexStart到indexEnd（不包含）的新字符串
    • 省略indexEnd，提取到字符串末尾

# 构造

  • padStart(targetLength, padString)
  • padEnd(targetLength, padString)

分别在字符串首、尾，将达不到指定长度targetLength的不用使用padString进行补足。

"abc".padStart(10); // "       abc"
"abc".padStart(10, "foo"); // "foofoofabc"
"abc".padStart(6, "123465"); // "123abc"
"abc".padStart(8, "0"); // "00000abc"
"abc".padStart(1); // "abc"
